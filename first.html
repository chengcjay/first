兩百字內的個人簡介

我是鄭勁節，目前是台大政治所碩班三年級的學生，預計將於下學期（ 2020 年 7 月 ）畢業，正在撰寫論文中，研究主題是臺灣的民族主義。
苗栗人，現居台北。國小、國中曾就讀美術班，高中編校刊；大學與研究所期間曾在圖書館、公關公司以及藝術電影院工讀，有良好的溝通、美術與寫作能力。
在畢業前夕，想在自己既有的專業基礎之外，多學一些技能，拓展視野，增加未來更多的可能性。

為什麼想成為工程師 ( 請以自我期許帶入：前端、後端、或全端工程師 )

最主要的考量是就業。大學與研究所學的政治學沒有直接對應的行業與職位，所以需要更多額外的涉略。而工程師是當代需求最大的專業之一，無論是哪個行業都離不開網站、軟體。以媒體產業為例，記者的待遇並不理想，但若是以工程師的角色參與其中，則既有滿意的薪資，又可兼顧對新聞媒體的憧憬與抱負。
在摸索、初學 python 的過程中，我從中感受到許多樂趣，加深了我學習程式語言、成為工程師的想法。因為有美術與刊物排版的經驗，所以我想以前端工程師為目標。

為了成為工程師，曾經做過什麼努力？

目前已把彭彭老師的 python 入門教學課程 29 部影片全部看完，並跟著練習、操作。雖偶有撞牆，但最後都能克服，找出自己的程式出錯的地方，並在過程中感受到成就感。

對於工程師的職業想像是什麼？

我認為工程師是一個需要不斷精進、學習的職業。因為社會、科技的進展十分迅速，工程師是走在最前端的人。對我而言，工程師的魅力在於能夠將腦海中的想法，經由程式語言與電腦將之實現，那是實驗與創造的過程。

如果參與這個計畫，會怎麼安排學習時間？

目前生活的重心是論文的寫作，該修的學分都已修完。主要的架構、大綱已經完成，下個學期就進入反覆的討論與修改。
因此，如果能夠參與這個計畫，我的時間將會是論文寫作與學習程式語言兩者相交替。最有可能的安排會是，上午學習程式語言，午後寫論文進度，晚飯後再繼續程式語言的課程、練習，直到當日進度達成。

如果參與這個計畫，預期會碰到什麼困難？你打算怎麼解決它？

如同彭彭老師在課程簡介的直播中說的，學習程式語言、成為工程師，時常會遇到學習瓶頸、信心低落的問題，我覺得這會是最可能的預期困難。一旦信心消磨的差不多時，就會降低學習的效率甚至會放棄學習。因此，我會不斷提醒自己得初衷，學習是為了畢業之後能有一份兼顧生活品質與理想的工作。畢業是壓力也會是最好的動力。

想要對我們說的事情？

在寫論文寫到煩悶、對未來有些徬徨的時候，在 youtube 上看到彭彭老師的教學頻道，帶著好奇邊看邊學，意外地有種安定感與成就感。在這裡要向彭彭老師說聲感謝，願意在網路上免費釋出這麼珍貴的資源。希望有機會能夠參與這門課程，成為一名工程師。
